# 活動照片牆 (Memento)

即時互動照片牆平台，讓活動參與者透過手機上傳照片和發送彈幕，即時顯示在大螢幕上。

## 特色功能

- 🖼️ **即時照片上傳**：參與者上傳的照片 3 秒內顯示在大螢幕上
- 💬 **彈幕留言**：發送文字留言，以彈幕形式飄過螢幕
- 🔐 **無需登入**：使用活動代碼或 QR Code 快速加入
- 🔒 **隱私優先**：零資料儲存，照片保存在參與者自己的 Google Drive
- ⚡ **高效能**：支援單一活動 500 並發使用者

## 快速開始

### 開發者設定（必要步驟）

在開始開發前，你需要：

1. **設定 Google Cloud 專案**：
   📖 [開發者設定指南](./GOOGLE_CLOUD_SETUP_DEV.md) - OAuth 2.0、API Key 設定

2. **使用者指南**（給活動主辦方）：
   📖 [Google Drive 設定指南](./GOOGLE_DRIVE_SETUP.md) - 如何建立和設定資料夾

